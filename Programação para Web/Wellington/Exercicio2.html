Exercícios

Estruturas de seleção simples e composta + exercícios (11/04/2017 - 11/04/2017

Faça uma versão utilizando condicional simples e outro com condicional


1 - Crie um script que avalie a escolha do usuário na utilização de uma caixa de diálogo de confirmação;



2 - Crie script que receba dois números e imprima o menor dos dois.



3 - Crie script que que receba um número, verifique se este número é par ou ímpar e imprima a mensagem. Para verificar se um número é par, use a operação resto da divisão por 2, se der zero, o número é par.


4 - Crie script que receba uma senha do usuário e compare com a senha padrão “ASDFG”. Se os valores forem iguais, o seu programa deve informar a mensagem “Acesso liberado”, em caso contrário, a mensagem a aparecer é “Acesso negado”.



5 - Crie script que receba quatro notas de um aluno, calcule e imprima a média aritmética das notas e a mensagem de aprovado para média superior ou igual a 7.0 ou a mensagem de reprovado para média inferior a 7.0.



6 - Crie script que receba a altura (H) e o sexo de uma pessoa, calcule e imprima o seu peso ideal, utilizando as seguintes fórmulas: para homens: (72.7 * H) - 58 para mulheres: (62.1 * H) - 44.7



7 - Crie script que receba um valor de 1 a 7 e mostre o dia da semana correspondente.
